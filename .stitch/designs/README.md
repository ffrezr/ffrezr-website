## Usage

These HTML files serve as the design source of truth. Open them in a browser to see the high-fidelity designs generated by Google Stitch. Use them as reference when building React components.

The design system follows a **High-End Editorial Portfolio** aesthetic with:
- Tonal architecture (#222222 primary, #F9F9F9 surface)
- Hairline borders (1px #C6C6C6)
- Manrope typography with tight letter-spacing
- Glassmorphism effects for floating elements
- Asymmetric layouts with generous negative space
